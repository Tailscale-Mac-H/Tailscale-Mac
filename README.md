## Overview

Tailscale Professional Suite is a complete mesh networking platform with subnet routing, exit nodes, and identity management for organizational network infrastructure.

Subnet routers expose entire network subnets to the Tailscale network, enabling access to devices that cannot run Tailscale directly — printers, IoT devices, legacy servers. The subnet router bridges the Tailscale mesh with traditional network infrastructure.

Exit nodes route internet traffic through a specific device on the network, enabling secure internet access through a trusted location when using untrusted networks.

SSO integration authenticates users through existing identity providers — Google, Microsoft, Okta — using existing organizational credentials rather than separate VPN credentials.

Tailscale Professional Suite is the mesh networking platform for organizations that need secure, zero-config private networking with the access control, subnet routing, and identity integration that enterprise networking requires.

## Performance & Integration

Tailscale is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Tailscale processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
